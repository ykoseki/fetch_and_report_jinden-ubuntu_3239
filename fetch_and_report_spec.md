# 業務改善報告書 自動生成スクリプト 仕様書 (fetch_and_report.py)

このドキュメントは、Pleasanter（プリザンター）からデータを取得し、月次レポートを自動生成するスクリプト `fetch_and_report.py` の仕様について記したものです。

## 1. スクリプトの概要

プリザンターの特定のサイト（ID: 3239）から指定した年月のレコードを取得し、以下の2つのファイルを自動生成します。
1.  **エクセルレポート (Excel)**: 2シート構成、書式・レイアウト調整済み。
2.  **お知らせ用テキスト (Text)**: 件数やURLが自動反映された通知文。

**★ 改善ポイント (2026/04/09 更新)**:
- **パスの自動検知**: 実行場所を問わず、スクリプトがあるフォルダを自動的に作業ディレクトリとして認識します。
- **Slim Edition (ポータブル設計)**: エクセルテンプレートやJSON定義ファイルがなくても、コード内部のデフォルト値を使用して動作可能です。
- **セキュリティの強化**: `python-dotenv` による環境変数の管理を導入し、APIキー等の秘密情報を安全に分離しました。 ✨
- **堅牢なログ出力**: 実行状況をブラケット形式（`[OK]`, `[ERROR]`等）で出力し、Windows等の日本語環境でも文字化けしにくい設計にしました。

## 2. 実行方法

Python環境に依存ライブラリをインストールした状態で、ターミナルから以下のコマンドを実行します。

```bash
# 基本実行 (2026年3月度を生成)
python fetch_and_report.py --year 2026 --month 3

# 引数なしで実行した場合、デフォルトで 2026/3 が対象となります
python fetch_and_report.py
```

### 依存ライブラリ
- `pandas`: データ処理
- `openpyxl`: エクセル書式・ページ設定の制御
- `python-dotenv`: 環境変数（.env）のロード ✨

```bash
pip install pandas openpyxl python-dotenv
```

## 3. 主要機能

### 3.1 データの動的取得とマッピング
- **API連携**: PleasanterサーバーのAPIを叩き、最新のユーザー名、組織名をリアルタイムで取得・統合します。
- **名称変換のフォールバック**: 外部のサイトパッケージ定義（JSON）がない場合でも、コード内に保持されたデフォルトの対応表（職場・ステータス等）を使用して名称変換を行います。

### 3.2 エクセルレポートの最適化
- **2シート構成**:
  - `高得点リスト(21点以上)`: 優秀事例のピックアップシート。
  - `全件リスト`: 該当月の全データ（フィルタなし）シート。
- **レイアウト制御**:
  - **列の非表示**: 指定された12項目（氏名, 部署名, 職場, 件名, 総合点数 等）のみを表示し、不要な計算用カラムを非表示にします。
  - **自動幅調整**: セル内の文字長（日本語全角対応）に基づき、カラム幅を自動で広げます。
  - **印刷設定**: A4横向き、横幅1ページ収まりに自動設定されます。

### 3.3 通知メッセージの生成
- **動的カウント**: 入賞者数と全件数を自動カウントし、定型文に流し込みます。
- **動的 URL 短縮**: 指定年月のフィルタがかかったプリザンター画面へのURLを動的に構築し、TinyURL API で短縮して出力します。
  - ※内部サーバー宛のリンクのため、TinyURLの画面で一度クリックが必要です。

## 4. ファイル構成

| ファイル名 | 役割 | 必須/任意 |
| :--- | :--- | :--- |
| `fetch_and_report.py` | メインプログラム（実行ファイル） | **必須** |
| `.env` | 接続設定（URL, APIキー, サイトID） | **必須** (セキュリティ環境) |
| `202603.xlsx` | エクセル出力用テンプレートファイル | 任意 (書式コピー用) |
| `*.json` | サイト定義（パッケージ）ファイル | 任意 (名称変換用) |
| `jinden_*.json` | マスタデータのキャッシュ | 自動生成 |

## 5. 注意事項
- 実行時に同名のエクセルファイルが開いていると、`PermissionError` が発生します。必ずファイルを閉じてから実行してください。
- プリザンターサーバー (`jinden-ubuntu`) へのネットワーク接続が必要です。
